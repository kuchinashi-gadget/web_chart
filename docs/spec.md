# Chart Playback 仕様メモ

## 目的

Chart Playback は、過去チャートを1本ずつ進めながら、売買判断・建玉管理・振り返りを練習するWebアプリ。

投資助言ではなく、チャート読解と売買判断を練習するための個人学習ツール。

## 対象データ

CSVを `public/data` に配置して読み込む。

基本列:

```csv
Date,Open,High,Low,Close,Volume
```

Volume は未使用でもよい。

対応想定:

- 日本株
- 指数
- 米国株
- 先物
- 為替
- 仮想通貨
- ETF

銘柄定義には以下を持つ。

- code
- name
- path
- currency
- unitLabel
- defaultLotSize
- multiplier

## チャート機能

- ローソク足表示。
- 移動平均線表示。
- 日足 / 週足 / 月足の切り替え。
- キーボード左右キー、画面ボタンで1本送り。
- マウスホイールや横スクロールで表示位置移動。
- 表示本数は以下から選択。
  - 毎回自動
  - 前回を引き継ぐ
  - 固定本数
- 初期表示位置は以下から選択。
  - 最新
  - 最新からN本前
  - 前回の続き
  - 最初の方
- 表示本数と右端日付は銘柄・足種ごとに保存。

## 足種切り替えの日付仕様

日足から週足・月足へ切り替える時は、基準日より未来の未完成足を表示しない。

例:

- 日足が 2020/6/4 の場合
  - 週足は 2020/5/29 まで
  - 月足は 2020/5/29 まで
- 日足に戻した時は、元の日足 2020/6/4 に戻る。

週足・月足側で移動した場合は、その足に含まれる最後の日足へ基準日を更新する。

## 売買練習

注文種別:

- 売りを追加
- 売りを返済
- 買いを追加
- 買いを返済

建玉表示:

```text
売玉 - 買玉
```

売買はFIFOで返済する。

約定タイミング:

- 次の取引日の始値
- 当日の終値

設定から選択できる。

## 損益計算

損益は以下で計算する。

```text
価格差 × 1玉あたり数量 × 玉数 × multiplier
```

日本株は基本的に:

```text
1玉 = 100株
currency = JPY
multiplier = 1
```

指数などは:

```text
1玉 = 1
unitLabel = ""
currency = JPY
multiplier = 1
```

## 売買ログ

売買ログには以下を表示する。

- 注文種別
- 玉数
- 数量
- 注文日
- 約定日
- 約定価格
- 確定損益
- 返済した建玉
- 勝敗

建玉ログと返済ログは、hover / click で対応関係をハイライトする。

削除仕様:

- 返済ログを削除した場合、その返済で閉じた建玉を復活する。
- 建玉ログを削除した場合、紐づく返済ログも一緒に削除する。
- 削除対象の返済ログに別の建玉が混ざっていた場合、その別建玉は復活する。

## 勝敗集計

返済ログを対象に勝敗を集計する。

- 損益率が引き分け基準未満: 引き分け
- それ以上の利益: 勝ち
- それ以上の損失: 負け

初期の引き分け基準:

```text
0.5%
```

設定画面から変更できる。

勝敗は損益と同じように表示/非表示を切り替えられる。

## チャートメモ

- チャート上のローソク足をクリックして対象日を選べる。
- メモはクリックした日付に紐づく。
- 文章だけのメモも表示される。
- 表示外にあるメモの対象日は、表示範囲の左端または右端に寄せて扱う。
- 今後、メモ色や入力UIをさらに改善予定。

## ペイント練習

ライブチャートに直接描画するのではなく、チャートをスクリーンショット化し、その画像に描画する。

現在できること:

- 現在のチャートをスクショして開始。
- 直線、曲線、フリーハンド、矢印、丸・楕円、四角、テキスト、消しゴム。
- 色選択、カスタム色。
- 太さ変更。
- PNGダウンロード。
- ペイント履歴保存。

改善予定:

- テキスト入力UI改善。
- Ctrl+Z対応。
- ツールごとの色履歴。
- テキスト色保持。

## 外観・設定

設定画面の主な項目:

- 移動平均線
- 表示
- チャート外観
- 売買設定
- 言語

言語:

- 日本語
- 英語

外観はチャートだけでなく、アプリ全体にテーマが反映される方向で調整中。

## 保存

ブラウザの localStorage / IndexedDB を使う。

保存対象:

- 売買練習データ
- チャートメモ
- チャート設定
- 表示本数・右端日付
- ペイント履歴

クラウド保存やユーザー登録は後回し。

## 注意

- 投資助言ではない。
- 損益計算は簡易計算。
- 手数料、税金、スプレッド、証拠金、先物や為替の取引単位などを完全には反映していない。
- 市場データの正確性、完全性、継続取得可能性は保証しない。
