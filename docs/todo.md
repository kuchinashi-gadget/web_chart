## CSV読み込み・銘柄管理

CSV読み込み・銘柄管理の無料版向け最小範囲はおおむね実装済み。
株価データの CSV をブラウザから読み込み、チャート表示できるようになった。

今後は公開前の整理、会員プラン別の制限、必要に応じた有料向け拡張を検討する。

CSV・銘柄管理に関係する部分は、全体リファクタではなく必要な範囲だけを最小限分割する方針。

### D-1. CSV・銘柄まわりの最小限ファイル分割

目的:

- App.tsx の肥大化を抑える。

- Codex の差分を小さくする。

保留:

保存処理は App 内の型や状態への依存が広いため、今回は無理に分割しない。

条件:

- 新機能はまだ追加しない。
- 既存の売買練習、チャートメモ、ペイント練習、本数計測は壊さない。
- `npm run build` と `npm run lint` が通ることを確認する。
- 変更が大きくなりそうなら、実装前に止めて確認する。

### D-2. ブラウザから CSV を一時読み込み

実施済み。

- ファイル選択で CSV を読み込む。
- `Date,Open,High,Low,Close` を検証する。
- `Volume` は任意。
- 読み込めたら一時銘柄としてチャート表示する。
- リロードしたら消えてよい。
- 既存の銘柄切り替えや売買練習を壊さない。

現在の仕様:

- ツールバーの `CSV読込` / `Import CSV` からファイルを選ぶ。
- ファイル名からコード・表示名を仮決定する。
- 価格の小数桁は CSV 内の価格から自動推定する。
- 通貨、1玉あたり数量、倍率などは仮の汎用設定で読み込む。
- 保存済み銘柄として登録した場合は、リロード後も利用できる。

### D-3. 読み込んだ CSV の銘柄情報設定

実施済み。

設定項目:

- 銘柄名
- コード
- 市場 / 種別
- 通貨
- 1玉あたり数量
- multiplier
- 表示名

日本株以外の指数・為替・先物・仮想通貨でも使えるようにする。

現在の仕様:

- CSV 読み込み直後に一時銘柄設定を開く。
- 一時銘柄を選択中はツールバーの歯車ボタンから再編集できる。
- 設定できる項目は、コード、銘柄名、通貨、単位、1玉あたり数量、倍率、価格小数桁。
- 設定は一時銘柄に即反映する。
- 保存済み銘柄として登録した場合は、リロード後も利用できる。

### D-4. 銘柄管理

#### D-4a. 読み込んだCSVを保存済み銘柄にする

実施済み。

- 一時CSV銘柄の設定画面から「銘柄一覧に保存」できる。
- 保存した銘柄は、リロード後も銘柄選択プルダウンに表示される。
- 保存済み銘柄を選択すると、IndexedDB に保存した CSV データからチャートを再表示する。
- CSV データ本体は IndexedDB に保存。
- 銘柄メタ情報は localStorage に保存。
- 一時銘柄で作成した売買練習データとチャートメモは、保存済み銘柄側へ引き継ぐ。

#### D-4b. 保存済み銘柄の編集・削除・CSV差し替え

実施済み。

- 保存済み銘柄を選択中も、ツールバーの歯車ボタンから設定を再編集できる。
- コード、銘柄名、通貨、単位、1玉あたり数量、倍率、価格小数桁を更新できる。
- CSV を差し替えて、同じ保存済み銘柄の価格データを更新できる。
- 保存済み銘柄を削除できる。
- 削除時は、その銘柄の売買練習データとチャートメモも削除する。
- 選択中の保存済み銘柄を削除した場合は、標準銘柄へ戻す。

#### D-4c. 銘柄管理タブ

実施済み。

- 設定ダイアログに「銘柄管理」タブを追加。
- 保存済み銘柄一覧を表示。
- 一覧から編集、CSV差し替え、削除ができる。
- 選択中の銘柄が分かるように表示。
- 銘柄ごとの売買ログ数とチャートメモ数を表示。
- 一時CSV銘柄を保存する際、同じコードの保存済み銘柄がある場合は警告する。
- 重複時は、既存銘柄へ上書き、別銘柄として保存、キャンセルを選べる。

## Pro / 真剣練習向けに移動

### 練習結果サマリー

通常練習では急いで実装しない。

真剣練習モードとセットで実装する。

通常練習は、過去に戻れる・銘柄を変えられる・ログ削除できるため、厳密な成績サマリーとの相性が弱い。

真剣練習モードでは以下を制限する。

- 時間を戻せない。
- 途中で銘柄を変えられない。
- 売買ログを削除できない。
- 練習終了時に結果サマリーを表示する。
- 練習履歴を保存する。

サマリー表示候補:

- 勝ち数
- 負け数
- 引き分け数
- 総損益
- 最大利益
- 最大損失
- 売買回数
- 練習期間
- 勝率
- 平均利益
- 平均損失
- 最大ドローダウン
